##  핵심 기능 및 구현 내용

###  1. GSAP 기반의 스토리형 타임라인 인터랙션 (UX 중심)
- 유저의 시선과 스크롤 흐름을 고려하여 자연스럽게 몰입할 수 있는 스토리텔링 인터랙션을 설계했습니다.
- **GSAP(GreenSock)** 라이브러리를 고도화하여 `밤 ➡️ 노을 ➡️ 낮 ➡️ 봄 ➡️ 들판 ➡️ 땅`으로 매끄럽게 연결되는 타임라인 시퀀스 애니메이션을 구현하여 시각적 즐거움과 독창성을 부여했습니다.

###  2. Grid & Tailwind CSS 기반의 반응형 레이아웃
- CSS Grid와 Tailwind CSS의 단축 문법 및 반응형 Breakpoints 사양을 적극 활용하여 모바일, 태블릿, 데스크톱 등 어떠한 디바이스 환경에서도 무너지지 않는 견고한 가변 레이아웃을 완성했습니다.

###  3. 컴포넌트 에코시스템 구축 (shadcn/ui 도입)
- 현대 프런트엔드 트렌드인 `shadcn/ui` 라이브러리를 도입하여 Radix UI 프리미티브 기반의 접근성(Accessibility)을 확보했습니다.
- 유연하면서도 재사용성이 극대화된 UI 컴포넌트 아키텍처를 수립하여 향후 새로운 프로젝트나 이력을 쉽게 확장·유지보수할 수 있도록 설계했습니다.

###  4. GitHub & Vercel 연동 기반의 CI/CD 파이프라인 구축
- 프로젝트 마스터 브랜치에 코드가 push되면 자동으로 빌드 및 테스트를 거쳐 실시간으로 결과물이 반영되는 **Automated Deployment(CI/CD)** 환경을 구축했습니다.
- 이를 통해 신규 프로젝트 개발 및 이력 업데이트 시 수동 배포 리소스를 제로화하고 지속적인 무중단 운영이 가능하도록 구성했습니다.
